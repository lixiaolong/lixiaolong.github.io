开始

2020-04-19

Hugo⌗

hugo是一个静态博客工具，采用我比较喜欢的golang编写

# 官网
https://gohugo.io

# 源码
https://github.com/gohugoio/hugo

# 下载
https://github.com/gohugoio/hugo/releases

可以从github的release下载，如果有搭建golang编译环境的话，也可以自行编译。

# 开启go module
go env -w GO111MODULE=on

# 编译并安装
go get -ldflags "-w -s"  github.com/gohugoio/hugo

# 注意添加GOPATH/bin到环境变量中，就可以使用hugo命令了

使用方法

# 创建站点
hugo new site raindrop

# 新建文章
hugo new posts/theme.md

# 开启本地服务，执行成功后就可以使用浏览器访问 http://localhost:1313
hugo server

Git⌗

git是一款版本控制软件，是目前最流行的代码管理软件，它可以追踪文件的改动，并方便的进行回退，所以我们这里用它来管理雨滴。

# windows下载地址
https://git-scm.com/

下载安装后，就可以通过如下命令来管理了。

cd raindrop
git init
git add *
git commit * -m"init raindrop"

Theme⌗

hugo要下载一个主题才能使用，官网上提供了大量主题可供选择，我这里选择的是一款名叫Terminal的主题，其设计是模仿终端命令行，界面简洁美观，是我喜欢的风格。

# 主题列表
https://themes.gohugo.io/

# Terminal
https://themes.gohugo.io/hugo-theme-terminal/

创建好站点之后，就可以下载主题了。

# 下载主题
git submodule add https://github.com/panr/hugo-theme-terminal.git themes/terminal

Github⌗

github是一个代码托管软件，我们可以把雨滴放到上面，这样离开了本地电脑也能访问，甚至github pages还可以提供免费的网站搭建。

首先要创建github账号，然后创建一个名为raindrop的空仓库，用来放我们的源码，再创建一个名为lixiaolong.github.io的空仓库，用来放最终的网站代码，其中lixiaolong是github的用户名。

# 之前已经创建了本地的raindrop项目，并且已经提交过了，所以这里直接指向远程仓库
git remote add origin https://github.com/lixiaolong/raindrop.git
git push -u origin master

# 添加子模块 lixiaolong.github.io
git submodule add https://github.com/lixiaolong/lixiaolong.github.io.git public

之后，直接执行hugo即可生成页面到public中，再提交所有即可。

最后将代码提交到远端，就可以浏览器访问了，注意主目录和public都要提交。

# 浏览器访问
https://lixiaolong.github.io
