用 QQ 播放器看斗鱼

:: lxl
#DIY  # 

概述⌗

无聊的时候就会看看斗鱼直播，主要看看游戏区，奈何网页和官方客户端上各种眼花缭乱的功能，看着就心烦意乱，Win10 上确实有个官方 UWP 的客户端又简洁又美观，然而官方却放弃了这个客户端，目前这个 UWP 版本播放时不时就闪退，实在抓狂，所以只得到网上找找有没有解决办法，于是在网上搜集到了一个比较符合我的口味的解决办法，在这里记录下来。

# 参考链接
https://www.52pojie.cn/thread-1096152-1-1.html

步骤⌗

第一步是一个关键网址

https://web.sinsyth.com/lxapi/douyujx.x?roomid=房间号

如果房间当前没有直播，则会返回未开播的json

{"type":"strjson","Rendata":{"tname":"Unbroadcast","time":1588657716},"info":"Live broadcasting has been closed","state":"NO"}

如果房间正在直播，则会返回直播的地址

{"type":"strjson","Rendata":{"media_type":"live","data":{"vid":"208114","roomName":"\u8bf7\u95ee\u8fd9\u91cc\u662f\u7089\u77f3\u533a\u58f0\u97f3\u6700\u5927\u7684\u76f4\u64ad\u95f4\u5417\uff1f","nickname":"\u6069\u57fa\u7231\u840c\u840c\u54d2\u7684\u72d7\u8d3c","avatar":"https:\/\/apic.douyucdn.cn\/upload\/avatar_v3\/201910\/54c8b6e4868041b1b1dc39776c110850_big.jpg?rltime","roomimg":"https:\/\/rpic.douyucdn.cn\/asrpic\/200505\/208114_1317.png\/dy1","roominfo":"\u4e0d\u7ba1\u4f60\u6765\u4e0d\u6765 \u6bcf\u5929\u6211\u90fd\u5728\u8fd9\u7b49\u4f60\u5440"},"link":"http:\/\/hdl1a.douyucdn.cn\/live\/208114rCvmwAkEjM_2000p.flv?wsAuth=5804929fa6cee67e690d1c6ef9f15eb6&token=cpn-androidmpro-0-208114-35b27a1378d7a7e3e8294e9b8cf5f0cb171e097595355a4e&logo=0&expire=0&did=3b261266e884c0ed89a49f0cb582dc30&origin=all&vhost=play2","time":1588656009},"info":"Check-OK","state":"SUCCESS"}

上面的 json 中，link 字段就是直播的地址，这里要做个变换。

http:\/\/hdl1a.douyucdn.cn\/live\/208114rCvmwAkEjM_2000p.flv?wsAuth=5804929fa6cee67e690d1c6ef9f15eb6&token=cpn-androidmpro-0-208114-35b27a1378d7a7e3e8294e9b8cf5f0cb171e097595355a4e&logo=0&expire=0&did=3b261266e884c0ed89a49f0cb582dc30&origin=all&vhost=play2

http:\/\/hdl1a.douyucdn.cn\/live\/208114rCvmwAkEjM_2000p

http:\/\/hdl1a.douyucdn.cn\/live\/208114rCvmwAkEjM

http://tx2play1.douyucdn.cn/live/208114rCvmwAkEjM

http://tx2play1.douyucdn.cn/live/208114rCvmwAkEjM.m3u8

最后这个就是直播源的地址，直接使用一款支持 URL 播放的视频播放器就可以，我这里用的是 QQ 播放器。

在 QQ 播放器打开在线 URL ，然后将上面的 URL 地址打开，即可开始播放直播视频了。

这种方法就只有视频，弹幕啥的都没有了。
